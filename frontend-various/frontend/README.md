# qifu4 frontend , build of svelte

This Svelte admin frontend build of svelte.

## Installation

```bash
pnpm i

```

## Backend API env value

Update your backend API value in `.env`:

```text
API_ENDPOINT=http://127.0.0.1:8088/api
```

## Start a local server

```bash
npm run dev
```

## How to build

```
npm run build
```
